Comment déclarer un tableau en javascript ?

En JavaScript, un tableau (ou array en anglais) est une structure qui permet de stocker plusieurs valeurs dans une seule variable.
Voici les différentes façons de déclarer un tableau en JavaScript :

  1. Déclaration simple avec crochets [ ] :
    ex:
    let monTableau = [];
    let nombres = [1, 2, 3, 4, 5];
    let couleurs = ["rouge", "bleu", "vert"];
  2. Déclaration avec le constructeur Array :

    Il est également possible de créer un tableau en utilisant le constructeur Array, mais ce n'est pas aussi courant que l'utilisation des crochets.

    Voici un exemple :
    let monTableau = new Array(5);

NB : Un tableau peut contenir des valeurs de types différents, par exemple des nombres, des chaînes de caractères, des objets, etc.

Quels sont les contextes d’utilisations de tableaux ?

Les tableaux en JavaScript sont utilisés dans plusieurs contextes car ils permettent de stocker et manipuler des collections de données de manière flexible. Voici quelques contextes courants d'utilisation des tableaux :

Stockage de données similaires :

Un tableau est souvent utilisé pour regrouper plusieurs éléments de même type (nombres, chaînes, objets, etc.) afin de les manipuler plus facilement.

Manipuler des collections d'objets :

Les tableaux sont souvent utilisés pour regrouper des objets dans des situations où chaque objet représente une entité plus complexe (comme un utilisateur, un produit, etc.).
